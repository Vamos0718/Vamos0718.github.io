Queue

Same as stack, queue is one type of data structure, too. It is a linear storage structure and could be divided into two categories, they are link implementation and array implementation. The biggest difference with queue and stack is that, queue has the character which is First-In-First-Out, opposite of LIFO. In queue, the endpoint which enqueues data element is called queue rear, and another endpoint which dequeues data element is called queue front.
